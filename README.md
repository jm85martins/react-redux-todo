# React app with redux

### Todo List App
Simple TODO List built with:
* yarn
* react
* redux
* redux devtools
* thunk
* json-server

This is part of the course "Build A React App With Redux - Course by @avanslaars" from egghead.io

#### Setup

run:

```shell
yarn install
```

#### Test app

```shell
yarn test
```

#### Start app

dev mode:

```shell
yarn dev
```

run mode:

```shell
yarn build
yarn start
```
